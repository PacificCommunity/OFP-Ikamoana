ikamoana.dymfiles.dymtodataarray module

This module generate xarray.DataArray from DYM files.

ikamoana.dymfiles.dymtodataarray.dym2ToDataArray(infilepath: str, varname: str, attributs: Optional[dict] = None) → xarray.core.dataarray.DataArray

Generate a xarray.DataArray from the DYM file which the filepath is infile.

Parameters

  • infilepath (str) – The path to the DYM file.

  • varname (str) – The name of the generated DataArray.

  • attributs (dict, optional) – Attributs to add to the generated DataArray, by default None.

Returns

The DataArray generated from the DYM file.

Return type

xarray.DataArray
